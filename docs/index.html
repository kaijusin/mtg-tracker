🎤 Voice Command Examples
💚 Life Changes (fuzzy match):
• "Kai loses a life" → Kai loses 1 life
• "Chi gains 5 life" → Kai gains 5 life (fuzzy match)
• "Mike loses 10 life" → Mike loses 10 life
🃏 Hand & Deck:
• "Kai draws 2 cards" → Kai draws 2, hand increases
• "Chi pulls a card" → Kai draws 1
• "Kai discards 3 cards" → Kai discards 3, hand decreases
⚔️ Combat Damage:
• "Kai attacks for 3" → Kai takes 3 damage
• "attacks Sarah with 5" → Sarah takes 5 damage
• "for 8 damage" → Current defender takes 8 damage
⏱️ Turn Phases:
• "next phase" / "pass phase" → Advance one phase
• "set phase to combat" → Jump to Combat phase
• "set phase to draw/main/end" → Jump to that phase
• "next turn" / "end turn" → Pass to next player's turn
⚔️ Combat Phases (during combat):
• "attack with [card]" → Auto-start combat and declare attacker
• "declare done" or "pass" → Finalize attackers
• "no blockers" → Skip blocker assignment
• "proceed" or "resolve" → Move to next phase
• "done" → Resolve combat and see results
☠️ Poison Counters:
• "Kai takes poison" → Kai gets 1 poison
• "Chi gets 2 poison" → Kai gets 2 poison
• "poison counters" → Current player gets 1
🎴 Card Actions:
• "Kai cast lightning bolt" → Add card to graveyard
• "Kai placed Sol Ring into battlefield" → Add permanent to battlefield
• "tap goblin" → Tap/untap a card
• "Kai gains 3 life" / "Kai loses 2 life" → Update life
• "next phase" / "set phase to combat" → Control turn flow hands-free
• "next turn" → Advance to next player's turn
• "undo" or "undo last" → Revert last committed action
💡 Pro Tips:
✓ Names use fuzzy matching (Kai ≈ Chi, Sarah ≈ Sara)
✓ Numbers are optional (defaults to 1)
✓ Try natural speech — the parser is forgiving!
✓ During combat, use "pass" or "proceed" to advance phases
✓ Use "undo" immediately if voice parses the wrong action
